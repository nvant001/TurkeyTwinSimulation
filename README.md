#TurkeyTwinSimulation: Supply Chain Digital Twin

##Overview
This is a simulation engine to optimize autonomous vehicle fleets in a supply chain.
Leverges DES, AI demand predictions, and real-time dashboarding.

## Tech Stack
- **Core:** Python 3.12
- **Data:** Pandas, SQLite
- **AI/ML:** Scikit-Learn
- **Dashboard:** Streamlit
- **DevOps:** Docker, GitHub Actions
